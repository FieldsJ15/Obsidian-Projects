# Python for Biomedical

## Core Stack
```
numpy / scipy      ← numerical foundation
matplotlib / plotly← visualization
pandas             ← tabular data
jupyter            ← exploration
```

## Imaging
```
pydicom            ← read/write DICOM → [[DICOM]]
SimpleITK          ← image processing, registration, segmentation
nibabel            ← NIfTI (brain imaging)
scikit-image       ← general image processing
```

## Neuroimaging & Biosignals
```
MNE                ← EEG/MEG → [[EEG]]
nilearn            ← fMRI analysis → [[MRI]]
mne-connectivity   ← brain connectivity
```

## Machine Learning on Medical Data
```
monai              ← medical imaging AI (segmentation, classification)
torch / tensorflow ← deep learning backbone
scikit-learn       ← classical ML
```

## Radiotherapy
```
pydicom (RT Plan)  ← [[MLC-Sim]]
pymedphys          ← medical physics toolkit
```

## App Building
```
FastAPI            ← REST backend
Streamlit          ← quick data apps
Gradio             ← ML demos
```
