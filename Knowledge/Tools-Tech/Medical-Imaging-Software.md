# Medical Imaging Software

## Open Source Tools

| Tool | Use | Format |
|------|-----|--------|
| **3D Slicer** | Full viewer, segmentation, registration | DICOM, NIfTI |
| **ITK-SNAP** | Segmentation (semi-auto) | NIfTI, DICOM |
| **FSL** | Brain analysis (fMRI, DTI) | NIfTI |
| **FreeSurfer** | Cortical surface reconstruction | NIfTI |
| **OHIF Viewer** | Web-based DICOM viewer | DICOM |
| **MNE** | EEG/MEG | EDF, FIF |
| **Fiji/ImageJ** | General image analysis | Most formats |
