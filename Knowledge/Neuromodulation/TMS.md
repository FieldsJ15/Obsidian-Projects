# TMS — Transcranial Magnetic Stimulation

## Mechanism
- Rapidly changing magnetic field → induces electric current in cortex
- Depolarizes neurons non-invasively through skull
- Coil types: figure-8 (focal), H-coil (deep TMS)

## Protocols
- **Single pulse** — research, mapping motor cortex
- **Paired pulse** — cortical excitability (SICI, ICF)
- **rTMS** — repetitive: 1Hz (inhibitory), 10Hz+ (excitatory)
- **TBS** — Theta Burst (cTBS inhibitory, iTBS excitatory) — faster protocol

## Clinical Uses
- **Depression** (FDA approved — iTBS protocol)
- OCD (deep TMS)
- Smoking cessation
- Stroke rehab (motor recovery)
- Pain (experimental)

## Research Uses
- Motor mapping before neurosurgery
- Causal brain-behavior relationships (virtual lesion)
- Combined TMS-EEG → [[EEG]]
- Combined TMS-fMRI → [[MRI]]

## Key Parameters
- Intensity: % Motor Threshold (MT)
- Frequency (Hz)
- Number of pulses
- Inter-train interval
- Target location (neuronavigation)
